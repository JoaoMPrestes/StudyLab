## [1.0.0] - 2025-11-19
### Adicionado
- Versão inicial do **StudyLab (AppQuiz)**.
- Suporte completo ao JavaFX.
- Estrutura Maven configurada.
- Sistema de arquivos JSON para armazenamento dos quizzes.
- Funcionalidade CRUD de perguntas e respostas.
- Navegação entre telas no JavaFX.
- Interface gráfica funcional.
- Sistema de pontuação e resultados.
- Instalações geradas para Windows.
- README profissional nos idiomas **PT-BR** e **EN-US**.
- Arquivo **LICENSE (MIT)** adicionado ao projeto.
- `.gitignore` totalmente configurado.
- Publicação completa do código no GitHub.

- **Depreciado**: para funcionalidades marcadas como obsoletas.
